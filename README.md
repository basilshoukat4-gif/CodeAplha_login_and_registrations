# CodeAlpha Login and Registration System

A simple Login and Registration System in C++ using File Handling.

## 🚀 Features

- User Registration
- User Login
- Duplicate Username Check
- File Storage System
- Validation System
- Authentication

## 🛠️ Technologies Used

- C++
- File Handling
- Functions
- ifstream
- ofstream
- Conditions

## 📌 How It Works

- User registers with username and password
- Credentials are stored in text files
- Login system verifies user identity
- Displays success or error messages

## ▶️ How to Run

1. Open CodeBlocks or Dev-C++
2. Paste the code
3. Compile and Run
4. Register and Login
